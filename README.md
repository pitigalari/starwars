## Star Wars | Android client

I've used the libraries/APIs below to build this app,

* [Android Navigation](https://developer.android.com/guide/navigation)
* [Android Paging 2](https://developer.android.com/topic/libraries/architecture/paging)
* [Dagger 2](https://dagger.dev/)
* [Glide](https://github.com/bumptech/glide)
* [Gson](https://github.com/google/gson)
* [OkHttp](https://square.github.io/okhttp/)
* [Retrofit](https://square.github.io/retrofit/)
* [RxJava 2](https://github.com/ReactiveX/RxJava)
* [Timber](https://github.com/JakeWharton/timber)
* [ViewBinding](https://developer.android.com/topic/libraries/view-binding)
